# Sentiment Analysis using LSTM

## Aim
To develop a sentiment analysis model using Deep Learning techniques to classify customer reviews into positive, negative, or neutral sentiments.

## Technologies Used
- Python
- TensorFlow / Keras
- LSTM (Long Short-Term Memory)
- NLP techniques

## Dataset
A custom dataset of customer reviews created for academic experimentation.

## Model
- Embedding Layer
- LSTM Layer
- Dense Softmax Layer

## Evaluation Metrics
- Accuracy
- Loss
- Confusion Matrix

## Conclusion
The LSTM-based sentiment analysis model successfully classified customer reviews and demonstrated the effectiveness of NLP and deep learning in real-world text classification tasks.
